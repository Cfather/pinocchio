You can choose either to install a pre-compiled release of Pinocchio or just build it directly from source. Here is a list of supported installation methods:

  • Linux
  • Mac OS X
  • pip
  • conda
  • ROS
  • Nix
  • Arch
  • Build with robotpkg
  • Build from source
